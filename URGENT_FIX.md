# 🚨 긴급 수정 필요!

## 현재 문제
- ✅ 서버 정상 작동
- ❌ 로그인 실패: "Unable to exchange external code"

## 🔴 100% 확실한 해결 방법

### 1단계: Supabase 대시보드 설정

**[클릭: Supabase 대시보드](https://supabase.com/dashboard/project/dlwuckbuhnfzmramxosg)**

1. 왼쪽 메뉴 → **Authentication**
2. 상단 탭 → **URL Configuration**
3. **Redirect URLs** 섹션 클릭
4. 아래 URL들을 **정확히** 입력 (복사-붙여넣기 권장):

```
http://localhost:3000/auth/callback
http://192.168.45.3:3000/auth/callback
```

5. **Site URL** 확인:
```
http://localhost:3000
```

6. **Save** 클릭! (중요!)

### 2단계: 카카오 개발자 콘솔 설정

**[클릭: 카카오 개발자 콘솔](https://developers.kakao.com/)**

1. 내 애플리케이션 선택
2. **플랫폼 설정** → Web
3. **사이트 도메인** 섹션에 다음 추가:

```
http://localhost:3000
http://192.168.45.3:3000
```

4. **저장**

### 3단계: 다시 로그인 시도

1. 브라우저에서 http://localhost:3000 접속
2. 로그인 버튼 클릭
3. 카카오 로그인 성공

---

## ✅ 성공 확인

- 홈으로 리디렉션됨

## ⚠️ 주의사항

### ❌ 하지 말아야 할 것
- URL 끝에 `/` 추가
- 공백 추가
- `https://` 사용 (로컬은 `http://`)
- 대소문자 혼동
- 저장 안 함

### ✅ 해야 할 것
- 정확히 위에 적힌 대로 입력
- 복사-붙여넣기 사용
- **Save/수정** 클릭!
- 완료 후 다시 로그인 시도

---

## 🔍 여전히 안 되면?

1. 브라우저 캐시 삭제 (Ctrl+Shift+Delete)
2. 시크릿 모드로 테스트
3. 다른 브라우저로 시도
4. 서버 재시작: Ctrl+C 후 `npm run dev`

자세한 내용: `SUPABASE_REDIRECT_URL_FIX.md` 참고
